# 🌮 Little Taco Shop

A multi-page static website built with HTML and CSS as part of my fullstack development journey following the [roadmap.sh](https://roadmap.sh) curriculum.

---

## 📋 Project Overview

Little Taco Shop is a fictional taco restaurant website designed to practice core HTML and CSS fundamentals. The site includes multiple pages, a contact form, a pricing table, and accessible, semantic markup throughout.

---

## 📄 Pages

- **Home** (`index.html`) — Main landing page with navigation, about section, menu table, and footer
- **Hours** (`hours.html`) — Store opening hours displayed in a clean, readable format
- **Contact Us** (`contact-us.html`) — Contact form and restaurant location details

---

## ✨ Features

- Semantic HTML5 elements (`header`, `nav`, `main`, `section`, `article`, `aside`, `footer`, `figure`)
- Responsive viewport meta tag for mobile compatibility
- Accessible form with labels, fieldset, legend, and input validation (`required`, `pattern`, `autocomplete`)
- Pricing table using `rowspan` and `colspan` for structured data
- Favicon for browser tab branding
- External CSS stylesheet linked across all pages
- Navigation linking between all pages
- In-page anchor links for smooth section jumping
- `Back to Top` link on each page
- Copyright footer with HTML entity `&copy;`
- External link to source citation opening in a new tab (`target="_blank"`)
- Collapsible content using `<details>` and `<summary>`
- Tooltip abbreviations using `<abbr>`
- Semantic time element using `<time datetime="">`
